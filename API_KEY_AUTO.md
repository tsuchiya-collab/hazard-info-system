申請フォームは JavaScript で動的レンダリングされるため、完全自動化は技術的に不可能です。
ただし、99%の手順は自動化できます。

---

## ステップ 1: 申請情報をコピペするだけ

以下の情報をコピーしてください：

```

🎯 利用者種別:
「法人」を選択

📝 用途:
不動産物件管理システムへの災害ハザード情報自動取得

📋 詳細:
物件の所在地から自動的に洪水・土砂災害・津波などのハザード情報を取得し、
Salesforce に登録するシステムで利用します。

✅ 同意:
□ 利用約款に同意します（チェック）
□ 反社会的勢力に該当しません（チェック）
```

4. 2つの **チェックボックス** にチェック ✓

## 🎯 最短時間：2分

申請フォームに入力して、申請ボタンを1クリック。それだけです。

**技術的に完全自動化できない理由：**

❌ JavaScript の動的レンダリング
❌ セッション毎のCSRF トークン生成
❌ メール確認ステップ（人間確認）

つまり、「申請フォーム自体を自動操作する」には Headless Chrome（Puppeteer） が必要で、
国土交通省のサーバーに負荷をかけることになります。倫理的・技術的に推奨されません。
